Face Recognition Login System

The Face Recognition Login System is a secure and efficient user authentication system using facial recognition technology. This project integrates a Flask API to allow real-time face data collection, training, and authentication. Users can log in seamlessly by verifying their identity with their facial features.

How It Works

Here’s how the system operates:

  • Collect Faces: Capture and store user facial data using a webcam.
  • Train Model: Train a facial recognition model using the collected face data.
  • Login: Authenticate users by matching their live facial data with the trained model.

GitHub Repository

You can find the source code for this project on GitHub. Follow the instructions in the README file to set up and run the project on your system.
